Weather App

A Weather application built while learning how APIs and frontend JavaScript work together in real projects. The app allows users to search any city and instantly view real-time weather information through a clean and responsive interface.

Features -

- Search weather by city name
- Real-time weather updates using API data
- Temperature, humidity, and weather condition display
- Dynamic weather icons
- Responsive and clean UI
- Error handling for invalid searches

Built using -

- HTML
- CSS
- JavaScript
- OpenWeather API

What I Learned -

This project helped me understand:

- How APIs work and how to fetch live data
- Asynchronous JavaScript using `fetch()`
- DOM manipulation and dynamic UI updates
- Handling user input and validation
- Debugging frontend issues independently
